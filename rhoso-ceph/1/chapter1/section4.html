Deploy the Ceph storage cluster

  1. Run the below command to bootstrap your Ceph Storage Cluster.

    cephadm bootstrap --apply-spec initial-config.yaml --mon-ip 192.168.51.211 --registry-json /etc/auth.json --allow-fqdn-hostname

You can access the Ceph CLI as following in case of multi-cluster or non-default config:

        sudo /usr/sbin/cephadm shell --fsid ac86af9a-2fdf-11ef-8c9c-52540001326f -c /etc/ceph/ceph.conf -k /etc/ceph/ceph.client.admin.keyring

Or, if you are only running a single cluster on this host:

        sudo /usr/sbin/cephadm shell

Please consider enabling telemetry to help improve Ceph:

        ceph telemetry on

For more information see:

        https://docs.ceph.com/en/latest/mgr/telemetry/

Bootstrap complete.
You may have to wait for approximately 5 to 10 minutes for all the background processes needed for installation to complete and the cluster to be in HEALTH_OK state. You may track the progress with watch caph -s command.
