Create ODF StorageSystem

  1. Click on the Refresh web console pop-up option on top right corner.

  2. When the page is refreshed, click on Create StorageSystem button.

    You may need to navigate to the Operators then Installed Operators and then click on OpenShift Data Foundation option to get the option to create storage system if you do not see it after page refresh.

  3. Select Create an external storage platform option and make sure Red Hat Ceph Storage option is selected, then click on Next button.

  4. Click on the Download script option on the next page.

  5. Access the lab outside the gui session to login to workstation vm as student user.

  6. Copy ceph-external-cluster-details-exporter.py script from workstation vm to utility vm.

    scp /home/student/Downloads/ceph-external-cluster-details-exporter.py lab@utility:/tmp/

  7. Login to utility vm as lab user.

    ssh lab@utility

  8. From there, copy the ceph-external-cluster-details-exporter.py file to cephservera vm.

    [lab@utility ~]$ scp /tmp/ceph-external-cluster-details-exporter.py root@cephservera:

  9. Login to cephservera as root user.

    ssh root@cephservera

  10. Run the script on cephservera with arguments --rbd-data-pool-name volumes.

    python ceph-external-cluster-details-exporter.py --rbd-data-pool-name volumes
    Sample output

  11. Exit from the cephservera and then from the utility vm’s session to be back on the workstation vm.

  12. Copy the json output generatedf by ceph-external-cluster-details-exporter.py script on cephservera and paste it in file.json on workstation vm.

    vi file.json

  13. Go back to the gui session of the workstation vm.

  14. Click Browse button under External storage system metadata and select file.json file and then click Next.

  15. Click on the Create StorageSystem button on the next page.
